# Awesome OSINT & Investigative Document Framework

A curated collection of advanced LLM prompts, forensic workflows, and data-mining structures designed for deep investigative journalism and documentary filmmaking.

## 🎯 Mission Statement
This open-source repository aims to democratize advanced data analysis for filmmakers and journalists. We are building a framework to analyze gray-market pharmaceutical data, verify cryptographic document anomalies, and process massive unindexed databases.

## 🧠 Why We Need Claude (Anthropic)
This project specifically requires Claude's 200k+ token context window. Standard LLMs hallucinate when parsing 100-page forensic PDFs. We are integrating Claude to maintain perfect recall across large-scale document analysis.
